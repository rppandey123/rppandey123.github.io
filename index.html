BMI Calculator: Features

The BMI Calculator

  • Automatically calculates BMI, or Body Mass Index
  • Suggests the statistical interpetation (normal,overweight etc.) corresponding to that BMI
  weight <- 90
  height <- 1.8
  bmi <-  27.7
  Status <- "Overweight"

BMI Calculator: Usage

Pretty straight forward

  • Enter the weight in kgs and height in metres.
  • Press on the click button

BMI Calculator: Audience

The application can be useful for following

  • Doctors
  • Individuals who want to track their weight
